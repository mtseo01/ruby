# Ruby PlayGround

<h3>1. Ruby Command Line Tool : To-Do</h3>

- parse Json file
- Create / Update / Read / Delete

<h3>Usage</h3>

- `$ ruby to_do_tool.rb [option]`
  - option :
    -  [-li] | [--list]   : View To-Do list
    -  [-c]  | [--create] : Create To-Do
    -  [-f]  | [--finish] : Change To-Do state to Done
    -  [-rm] | [--remove] : Remove To-Do from list
